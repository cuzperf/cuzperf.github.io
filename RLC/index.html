RLC 串并联合集

最后更新于：2023年3月29日 上午

同一类型元件串并联

类型 串联 并联
电阻 \(R_1 + R_2\) \(\frac{1}{\frac{1}{R_1} + \frac{1}{R_2}}\)
电感 \(L_1 + L_2\) \(\frac{1}{\frac{1}{L_1} + \frac{1}{L_2}}\)
电容 \(\frac{1}{\frac{1}{C_1} + \frac{1}{C_2}}\) \(C_1 + C_2\)

电感电容离散化

梯形法

\[ u = L \frac{di}{dt} \stackrel{Trapezoid}{\Longrightarrow} \frac{u(k) + u(k - 1)}{2} = L \frac{i(k) - i(k - 1)}{\Delta t} \] 电感离散形式：\(i(k) = \frac{\Delta t}{2L} u(k) + \frac{\Delta t}{2L} u(k) + i(k - 1)\)

\[ i = C \frac{du}{dt} \stackrel{Trapezoid}{\Longrightarrow} \frac{i(k) + i(k - 1)}{2} = C \frac{u(k) - u(k - 1)}{\Delta t} \]

电容离散形式：\(i(k) = \frac{2C}{\Delta t} u(k) - \frac{2C}{\Delta t} u(k - 1) - i(k - 1)\)

梯形法 电感 电容
G \(G_L = \frac{\Delta t}{2 L}\) \(G_C = \frac{2C}{\Delta t}\)
j 初值 \(j(k) = G_L u(k) + i(k)\) \(j(k) = -G_C u(k) - i(k)\)
j 更新（串联） \(j(k) = 2 i(k) - j(k - 1)\) \(j(k) = -2 i(k) + j(k-1)\)
j 更新（并联） \(j(k) = 2 G_L u(k) + j(k - 1)\) \(j(k) = -2G_C u(k) - j(k - 1)\)

由于默认使用节点法求解，所以 j 的更新一般用 “并联” 方式(下同)

（后向 backward）欧拉法

\[ u = L \frac{di}{dt} \stackrel{Euler}{\Longrightarrow} u(k) = L \frac{i(k) - i(k - 1)}{\Delta t} \]

电感离散形式：\(i(k) = \frac{\Delta t}{2L} u(k) + i(k - 1)\)

\[ i = C \frac{du}{dt} \stackrel{Euler}{\Longrightarrow} i(k) = C \frac{u(k) - u(k - 1)}{\Delta t} \]

电容离散形式：\(i(k) = \frac{2C}{\Delta t} u(k) - \frac{2C}{\Delta t} u(k - 1)\)

半步长 欧拉法 电感 电容
G \(G_L = \frac{\Delta t}{2 L}\) \(G_C = \frac{2C}{\Delta t}\)
j 初值 \(j(k) = i(k)\) \(j(k) = -G_C u(k)\)
j 更新（串联） \(j(k) = i(k)\) \(j(k) = -i(k) + j(k - 1)\)
j 更新（并联） \(j(k) = G_L u(k) + j(k - 1)\) \(j(k) = -G_C u(k)\)

注意到梯形法和半步长欧拉法离散化对应的电感和电容是一致的。不同的仅仅是 j 的取值和迭代方式

RLC 电路

我们始终有 \(i(k) = Gu(k) + j(k - 1)\)

无论是电感还是电容或者是混合情况，甚至是电阻（电阻可以认为 \(j(k - 1)\) 恒等于 0）都成立

串联情况

\[ i = G_R u_{ab} = G_L u_{bc} + j_L = G_C u_{cd} + j_C \]

可知 \[ u = u_{ad} = u_{ab} + u_{bc} + u_{cd} = \frac{i}{G_R} + \frac{i - j_L}{G_L} + \frac{i - j_C}{G_C} \]

写成 \(i = G u + j\) 的形式可知

\[ G = \frac{1}{\frac{1}{G_R} + \frac{1}{G_L} + \frac{1}{G_C}} \qquad j = \frac{G}{G_L}j_L + \frac{G}{G_C}j_C \]

但是注意到这里的 \(j\) 的更新需要

并联情况

\[ i_R = G_R u, \quad i_L = G_L u + j_L, \quad i_C= G_C u + j_C \]

可知

\[ i = i_R + i_L + i_C = (G_R + G_L + G_C) u + (j_L + j_C) \]

\(j\) 的初值和更新

由于 RLC 串联有相同的电流 \(i\)，并联有相同的电压 \(u\)，因此 \(j\) 的更新方式要分情况讨论。另外 RLC 中 \(j\)的更新 必须依赖 \(j_L\) 和 \(j_C\) 的更新（即它们的值需要被保存）

串联方式更新

  1. 电流 \(i(k) = G u(k) + j(k - 1)\)
    • 梯形法：更新历史值 \(j_L(k) = 2 i(k) - j_L(k - 1), \; j_C = -2i(k) + j_C(k - 1)\)
    • 欧拉法：更新历史值 \(j_L(k) = i(k), \; j_C(k) = -i(k) + j_C(k - 1)\)
  3. 更新 \(j = \frac{G}{G_L}j_L + \frac{G}{G_C}j_C\)
  • 梯形法（更为复杂不用于计算）：\((\frac{G}{G_L} - \frac{G}{G_C}) i(k) + \frac{G}{G_L} j_L(k - 1) + \frac{G}{G_C} j_C(k - 1)\)
  • 欧拉法（更为复杂不用于计算）：\((\frac{G}{G_L} - \frac{G}{G_C}) i(k) + \frac{G}{G_C} j_C(k - 1)\)

并联方式更新

    • 梯形法：更新历史值 \(j_L(k) = 2 G_L u(k) + j_L(k - 1), \; j_C = -2 G_c u(k) - j_C(k - 1)\)
    • 欧拉法：更新历史值 \(j_L(k) = G_L u(k) + j_L(k - 1), \; j_C(k) = -G_C u(k)\)
  2. 更新 \(j = \frac{G}{G_L}j_L + \frac{G}{G_C}j_C\)
  • 梯形法（没有一般性不用于计算）：\(\frac{G}{G_L} j_L(k - 1) - \frac{G}{G_C} j_C(k - 1)\)
  • 欧拉法（没有一般性不用于计算）：\(\frac{G}{G_L} j_L(k - 1)\)

一直串联并联合并节点岂不是最终只有两个节点了。。。就变成了 Thevenin 等价定理了

RL 电路 和 RC 电路

考虑到电阻的特殊性，此类电路历史值可以直接更新而不依赖于原始的 \(j_L\) 或 \(j_C\) 的更新，又考虑到没有一般性，且并联不减少节点的个数，此处只给出串联的情况，思路都是一样的利用 \(u_L(k) = u(k) - \frac{i(k)}{G_R}\) 或 \(u_R(k) = u(k) - \frac{i(k)}{G_R}\)

从而

类型 梯形法 欧拉法
RL \(j(k) = (1 + k_{RL})G u(k) + k_{RL} j(k - 1)\) \(\frac{G}{G_L} Gu(k) + \frac{G}{G_L} j(k - 1)\)
RC \(j(k) = -(1 + k_{RC}) G u(k) - k_{RC} j(k - 1)\) \((\frac{G}{G_R} - 1) G u(k) + \frac{G}{G_R} j(k - 1)\)

其中 \(k_{RL} = \frac{G}{G_L} - \frac{G}{G_R}, \quad k_{RC} = \frac{G}{G_C} - \frac{G}{G_R}\),

草稿

梯形法 \[ \begin{aligned} j(k) &= \frac{G}{G_L} j_L(k) = G u_L(k) + \frac{G}{G_L} i(k) = G (u(k) - \frac{i(k)}{G_R}) + \frac{G}{G_L} i(k) \\ &= G u(k) + (\frac{G}{G_L} - \frac{G}{G_R}) i(k) = (1 + \frac{G}{G_L} - \frac{G}{G_R}) G u(k) + (\frac{G}{G_L} - \frac{G}{G_R}) j(k - 1) \end{aligned} \]

\[ \begin{aligned} j(k) &= \frac{G}{G_C} j_C(k) = -G u_C(k) - \frac{G}{G_C} i(k) = -G (u(k) - \frac{i(k)}{G_R}) - \frac{G}{G_C} i(k) \\ &= -G u(k) - (\frac{G}{G_C} - \frac{G}{G_R}) i(k) = -(1 + \frac{G}{G_C} - \frac{G}{G_R}) G u(k) - (\frac{G}{G_C} - \frac{G}{G_R}) j(k - 1) \end{aligned} \]

欧拉法

\[ j(k) = \frac{G}{G_L} j_L(k) = \frac{G}{G_L} i(k) = \frac{G}{G_L} Gu(k) + \frac{G}{G_L} j(k - 1) \]

\[ j(k) = \frac{G}{G_C} j_C(k) = - \frac{G}{G_C} G_C u_C(k) = -G(u(k) - \frac{i(k)}{G_R}) = (\frac{G}{G_R} - 1) G u(k) + \frac{G}{G_R} j(k - 1) \]

可以验证 \(R \to 0\) 时，对应 \(L, C\) 刻画一致

单步法通用推广

应该写成更通用的方式：不在关心里面到底是个啥，推广成单步法求解，总有 \(i(k) = G u(k) + j(k - 1)\), 其中 \(j(k) = a G u(k) + b i(k)\)

我们只需考虑两个组件的串联和并联情况即可，无论它是梯形法、欧拉法亦或是其它单步法，无论它是电阻、电感、电容、它们的并联串联混合都可以放在一起考虑。

设有两个端口网络分别满足 \[ i_1(k) = G_1 u_1(k) + j_1(k - 1), \; j_1(k) = a_1 G_1 i_1(k) \] 和 \[ i_2(k) = G_2 u_2(k) + j_2(k - 1), \; j_2(k) = a_2 G_2 i_2(k) \]

串联情况，即 \(i_1(k) \equiv i_2(k)\)

\[ u(k) = u_1(k) + u_2(k) = (\frac{1}{G_1} + \frac{1}{G_2}) i(k) - \frac{1}{G_1} j_1(k - 1) + \frac{1}{G_2} j_1(k - 1) \]

即 \(G = \frac{1}{\frac{1}{G_1} + \frac{1}{G_2}}\)

\[ j(k) = \frac{G}{G_1}j_1(k) + \frac{G}{G_2} j_2(k) = G (a_1 u_1(k) + a_2 u_2(k)) + (\frac{G}{G_1} b_1 + \frac{G}{G_2} b_2) i(k) \]

并不能写成原来的形式！失败！

因此模块合并一般并不能改变 j 的更新的个数！老老实实按照 \(j = \frac{G}{G_1} j_1(k) + \frac{G}{G_2} j_2(k)\) 来更新吧，或者叫分层更新法（更新构成有向图，可以拓扑排序后更新）。

并联情况，即 \(u_1(k) \equiv u_2(k)\)

\[ i(k) = i_1(k) + i_2(k) = (G_1 + G_2) u(k) + j_1(k - 1) + j_2(k - 1) \]

即

\[ j(k) = j_1(k) + j_2(k) = (a_1 + a_2) G u(k) + (b_1 i_1(k) + b_2 i_2(k)) \]

\(j\) 的更新与串联方式一致
